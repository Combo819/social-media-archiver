## Getting Started
1. Clone the repository (or click on the Gitpod button above)
```
git clone https://github.com/Combo819/social-media-archiver.git
```
2. Install Lerna
```
npm install -g lerna
```
3. install dependencies
```
cd social-media-archiver
lerna bootstrap
```
4. start
```
npm run start
```
